# GoLang OneShot

Simple, single-file examples to learn or review Go basics.

Each file covers one topic with comments and runnable code.

### Files

- vars.go        → Variables and declarations
- dtypes.go      → Basic data types
- arrays.go      → Arrays
- slices.go      → Slices
- maps.go        → Maps
- pointers.go    → Pointers
- structs.go     → Structs
- if-else.go     → If, else
- loops.go       → For loops (only loop in Go)
- functions.go   → Functions, returns
- errors.go      → Errors
- Interfaces.go  → Interfaces and type assertions
